## Abstract

StudyWithAI is a web application that helps students study more efficiently using AI. Instead of spending time creating notes or practice questions, users can enter a topic and the system will automatically generate flashcards and quiz questions.

The goal of this project is to make studying faster and more interactive. Students can review concepts using flashcards and then test their understanding through quizzes generated by AI. The application includes a secure login system so users can create accounts and access the study tools whenever they need them.

## How to Use the Application

1. Clone the repository to your PC.
2. Navigate to the `server` folder and run `npm install`.
3. Create a `.env` file and add your `MONGO_URI`, `JWT_SECRET`, and `OPENAI_API_KEY`.
4. Start the server using `npm start`.
5. Open your browser and go to `http://localhost:5050`.
6. Sign up or log in to your account.
7. Enter a study topic to generate AI flashcards and view your results on the dashboard.
