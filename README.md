StudyWithAI :
StudyWithAI is a web application that helps students study more efficiently using AI. Instead of spending time
creating notes or practice questions, users can enter a topic and the system will automatically generate flashcards and quiz questions.
The goal of this project is to make studying faster and more interactive. Students can review concepts using flashcards and then test their
understanding through quizzes generated by AI.
The application includes a secure login system so users can create accounts and access the study tools whenever they need them. Also feature that shows quiz performance statistics for users over time. 

Features : 
Secure authentication using JWT tokens
Password encryption using bcrypt
AI-generated flashcards
AI-generated quizzes
MongoDB database for storing user accounts and data
Backend API for generating study material

Technologies using : 
Node.js
Express.js
MongoDB Atlas
Mongoose
OpenAI API
JWT Authentication
HTML
CSS
JavaScript

Installation :
Clone the repository
Navigate into the project folder
Install dependencies
Create a .env file in the root folder (this folder has our private keys)


To use application: 
Start the server by running node server.js
Once the server is running, open your browser and go to the application.
